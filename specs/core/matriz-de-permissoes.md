# Matriz de Permissões

## Aluno

Permissões:

- criar conta
- realizar login
- editar perfil
- entrar em turmas
- assistir aulas
- acessar materiais
- baixar anexos
- participar de chats
- enviar mensagens
- responder questionários
- enviar atividades
- visualizar notas
- visualizar progresso
- conversar em mentorias privadas

Restrições:

- não pode criar cursos
- não pode criar turmas
- não pode moderar chats
- não pode suspender usuários

---

## Mentor

Permissões:

- todas as permissões de aluno
- responder dúvidas
- auxiliar alunos
- moderar conversas
- silenciar alunos do chat
- publicar resumos
- publicar materiais complementares
- acompanhar dificuldades dos alunos

Restrições:

- não pode criar cursos
- não pode remover professores
- não pode administrar o sistema globalmente

---

## Professor

Permissões:

- criar cursos
- editar cursos
- remover cursos
- criar turmas
- editar turmas
- remover turmas
- publicar aulas
- remover aulas
- criar questionários
- criar atividades
- corrigir atividades
- atribuir notas
- moderar chats
- silenciar usuários do chat
- suspender usuários do chat
- promover mentores
- remover mentores
- visualizar analytics da turma

---

## Administrador

Permissões:

- gerenciar usuários
- remover contas
- redefinir permissões
- moderar toda a plataforma
- suspender usuários da plataforma
- remover conteúdos inadequados
- visualizar relatórios globais
- administrar pagamentos
- administrar planos
- acessar analytics globais
- alterar configurações do sistema
